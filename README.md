# ThriftTest

### Thrift sample program for node.js (v0.10.30)

Preparation
```
> rm -rf node_modules
> npm install
> make thrift
```

#### test
```
npm test
```

#### module
| module |  func | desc |
| ---------- | ---------- | ---------- |
| serialzer.js | serialize | serialize ThriftObject to binary |
| | deserialize | deserialize ThriftObject from binary |
